# ✨ ESP32 OLED Stardust Animation

This project creates a dynamic "falling stars" animation on an OLED display using ESP32. It features custom bitmap rendering and randomized movement logic to simulate a cosmic atmosphere.

## 🚀 Features / Özellikler
* **Hardware:** ESP32 & SH1106 OLED Display (I2C).
* **Dynamic Logic:** Each star has a randomized horizontal position and vertical speed.
* **Bitmap Art:** Custom star shapes rendered using the Adafruit GFX library.
* **Multi-Language:** Documentation and code comments in both Turkish and English.

---

## 💻 Usage / Kullanım

1. Clone this repository: 
   `git clone https://github.com/AysenurKrkck/esp32-oled-stardust.git`
2. Open the `.ino` file in Arduino IDE.
3. Select your ESP32 board and the correct COM port.
4. Upload and watch the stars fall! ✨

---

## 📖 Project Structure / Proje Yapısı
- **`NUMFLAKES`**: Adjust this to change the number of stars.
- **`DELTAY`**: Controls the speed of the animation.
- **`logo16_glcd_bmp`**: Contains the HEX data for the star icon.
